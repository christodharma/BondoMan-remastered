# BondoMan: Remastered

Project - BondoMan in Flutter

## Overview

This is a remaster project from former BondoMan, a university project about transactions journal
built in Kotlin.\
The original projects has features like so:

1. Adding transactions, saving transactions into local storage
2. Looking back through transactions history
3. Camera access
4. Expense/Income visualization
5. Exporting .xls/.xlsx data
6. Sending exported data through email
7. Authorization (Login/Logout, given by database admin)

In this project, there are some twists on the features. And those features are:

| Original Feature              | Twist                    |
|-------------------------------|--------------------------|
| Admin-provided Backend (Auth) | Self-configured Supabase |
| Local Database                | Supabase Database        |
| And more coming...            |                          |
